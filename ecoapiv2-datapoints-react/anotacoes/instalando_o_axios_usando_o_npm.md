# Instalando o Axios usando o npm

https://www.npmjs.com/package/axios

1. Abra o terminal ou a linha de comando.

2. Navegue até o diretório do seu projeto usando o comando cd.

3. Execute o comando `npm init -y` para inicializar um novo projeto Node.js (caso ainda não tenha sido feito).

4. Instale o Axios executando o comando npm install axios.

5. Verifique a instalação consultando a seção de dependências no arquivo package.json. Você deverá ver uma entrada como "axios":"^<versão>".
